# BloodLink 🩸

BloodLink is a web-based emergency blood donor network that helps connect blood donors, seekers, and hospitals quickly during critical situations.

The platform enables users to search for compatible blood donors based on blood group and location, helping reduce the time required to find blood during emergencies.

---

## Features

### Donor Registration

* Register as a blood donor.
* Store donor details including blood group, city, and contact information.

### Smart Donor Search

* Search donors based on blood group.
* Filter donors by location for faster matching.

### Hospital Coordination

* Hospitals can access donor information during emergencies.
* Streamlined communication between hospitals and donors.

### Eligibility Tracking

* Tracks donor availability using last donation records.
* Prevents requests from being sent to temporarily ineligible donors.

### Emergency Request Management

* Create urgent blood requests.
* Match requests with suitable donors.

## Problem Statement

Blood shortages during emergencies often occur because finding compatible donors is slow and fragmented.

BloodLink aims to reduce this delay by providing a centralized platform that enables rapid donor discovery and efficient coordination between donors, seekers, and hospitals.
